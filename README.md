This project is based on the concept of open-source software and its real-world implementation. In this audit, Python has been studied in a Linux environment.

The aim of this project is to understand:
- How open-source software works
- The role of licensing
- How Linux supports open-source tools
- Practical usage through shell scripting

Along with theoretical understanding, five shell scripts have been created to perform different system-related tasks.

---



This script displays basic system details such as kernel version, user name, uptime, and date.


This script checks whether Python is installed and shows its version.


This script checks important system directories and displays their size, permissions, and ownership.


This script reads a log file and counts how many times a specific keyword (like "error") appears.
